# 語言處理
S = NP VP               名詞子句 + 動詞子句     
NP = DET Adj* N PP*     定詞 + 名詞     
VP = V NP               動詞 + 名詞子句     
PP = P NP               副詞 + 名詞子句

使用這種想法 一個一個慢慢生出語句
